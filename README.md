# 🎬 API de Filmes

Projeto desenvolvido para praticar conceitos de **Back-end com Node.js e Express**, simulando uma API simples para cadastro e consulta de filmes.

A API permite listar filmes, buscar um filme específico pelo ID e cadastrar novos filmes.

---

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- JavaScript
- Postman (para testes de API)

---

## 📌 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento **Back-end**, incluindo:

- Criação de APIs REST
- Uso de rotas com Express
- Utilização de verbos HTTP
- Uso de middleware
- Parâmetros de rota
- Testes de endpoints utilizando Postman
